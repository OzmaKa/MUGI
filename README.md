🌿 MUGI CSS Framework

MUGI is a lightweight, flexible CSS framework designed to give developers ready-to-use components without locking them into rigid styles or utility-only workflows.

Unlike heavy frameworks that fight your custom CSS, MUGI is built to be easy to understand, easy to override, and easy to extend. It provides structured, semantic components—such as buttons, navbars, footers, and layouts—while keeping styling transparent and developer-friendly.

✨ Why MUGI?

🧩 Prebuilt components (buttons, navbars, footers, and more)

🎨 Override-friendly by design — no specificity wars

🪶 Lightweight & minimal — no unnecessary bloat

🧠 Pure CSS first — JavaScript only when truly needed

🛠️ Component-based, not utility-only (not Bootstrap, not Tailwind)

MUGI sits between rigid frameworks like Bootstrap and utility-heavy ones like Tailwind—offering structure without sacrificing freedom.

Whether you’re building a small website or a scalable UI system, MUGI lets you start fast and still keep full creative control.
